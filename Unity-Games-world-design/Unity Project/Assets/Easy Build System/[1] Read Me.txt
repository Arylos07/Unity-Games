How To Works Out Of The Box ?

Before all, make a backup of your project this is very important.
To export the system (Out of box), it is necessary not to use a version of unity later than version 2017f.1 as well as the API Updater.
Asset has need imported directly from Asset Store on your project without the Project Settings.
If you have problems / questions during the operation, do not hesitate to contact us.

----------------------

How Fracture Your Own Models ?

The models available in this package have been fractured thanks to the Fracturing & Destruction asset (https://www.assetstore.unity3d.com/en/#!/content/9411).
This asset can be very useful if you want to break your own model or use fracturing algorithms on Maya or Blender.
